# FPL Podcast

## Description

This is a [FPL Podcast](https://audioboom.com/channels/5001585-fpl-pod) analysis system.
It works as follows

1. *Manually* get new podcast details from [RSS feed](https://audioboom.com/channels/5001585.rss)
2. Download new podcasts from [Audioboom](https://audioboom.com/)
3. *Manually* transcribe the podcasts using [Alice](https://aliceapp.ai/)
4. Use OpenAI to extract player based summary out of the transcripts
5. *Manually* update the player summaries to my website
